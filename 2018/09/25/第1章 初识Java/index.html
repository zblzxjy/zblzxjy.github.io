第1章 初识Java

Sep 25, 2018Java点击

“万丈高楼平地起，盘龙卧虎高山地”，可能用这句话来形容初学Java时的状态是比较恰当的。

1 计算机语言

1.1 什么是计算机语言？

计算机语言是人与计算机之间交流的工具

1.2 分类

1.2.1 机器语言

通常使用二进制命令进行编写

- 优点：可以直接执行，速度快，资源占用少

- 缺点：可读性、移植性差，编程复杂

1.2.2 汇编语言

用助记符代替二进制命令

- 优点：目标代码简短，占内存少，执行速度快

- 缺点：难学难写难记忆

1.2.3 高级语言

更符合人们的思维习惯
- 比如：Java、python、PHP等

2 走进Java

2.1 Java起源

Java是由James Gosling所创造，因此，他也被后辈称为：Java之父。

2.2 发展历程

- 1995年5月 Java语言诞生

- 1996年1月 JDK1.0发布

- 1999年6月 SUN公司发布Java三个版本

- 2000年5月 JDK1.3发布

2.3 三大版本

1. JavaSE（Java Platform Standard Edition）-标准版

- 用于开发和部署桌面、服务器以及嵌入设备和实时环境中的Java运用程序，用于开发电脑上运行的软件。

2. JavaEE（Java Platform Enterprise Edition）-企业版

- 简化企业解决方案的开发、部署和管理相关的复杂问题的体系结构，是用于网站开发。

3. JavaME（Java Platform Micro Edition）-微型版

- 为移动设备提供了Java环境的开发与应用平台，是用来开发手机软件的。

2.4 两大组件

1. JVM（Java Virtual Machine）-Java虚拟机


2. API（Application Programming Interface）

- 应用程序编程接口

3 初识Java

3.1 什么是Java

Java是一门面向对象编程语言，不仅吸收了C++语言的各种优点，还摒弃了C++里难以理解的多继承、指针等概  
念，因此Java语言具有功能强大和简单易用两个特征。

Java语言作为静态面向对象编程语言的代表，极好地实现了面向对象理论，允许程序员以优雅的思维方式进行复  
杂的编程。

3.2 Java语言特点

3.2.1 简单性

1. 由c++衍生而来，抛弃了C++语言的一些不是绝对必要的东西（比如：指针、结构等），更为简单

2. Java提供了丰富的类库，使得开发变得方便

3.2.2 面向对象

万物皆对象

3.2.3 平台无关性

Java虚拟机（Java Virtual Machine），对于不同的运行平台，有不同的虚拟机，屏蔽底层平台的差异，“一次编译，随处运行

3.2.4 健壮性

摒弃了指针和释放内存等C++功能，避免了非法内存操作。

3.2.5 安全性

通过Java的安全体系架构来确保Java代码的安全性

3.2.6 多线程

多线程机制使应用程序能够并行执行 ,而且同步机制保证了对共享数据的正确操作。通过使用多线程,程序设计者  
可以分别用不同的线程完成特定的行为 ,而不需要采用全局的事件循环机制 ,这样就很容易地实现网络上的实时  
交互行为。

3.2.7 面向对象

Java语言的设计集中于对象及其接口 ,它提供了简单的类机制以及动态的接口模型。对象中封装了它的状态变量  
以及相应的方法 ,实现了模块化和信息隐藏;而类则提供了一类对象的原型 ,并且通过继承机制 ,子类可以使用  
父类所提供的方法 ,实现了代码的复用。

3.2.8 垃圾回收机制

垃圾回收可以有效的防止内存泄露，有效的使用空闲的内存。

4 JDK和JRE

4.1 JRE

Java Runtime Environment，Java运行环境，缩写JRE

- 是运行Java程序所必须环境的集合，包含JVM标准实现及Java核心类库。

4.2 JDK

Java Development Kit，Java开发工具，缩写JDK

- 是Java语言的软件开发工具包(SDK),包含JRE

环境变量配置

- JAVA_HOME D:\Program Files\Java\jdk1.8.0_161

- CLASSPATH .;%JAVA_HOME%\lib

- Path %JAVA_HOME%\bin

- 若javac命令使用无效，则在用户变量中配置Path变量（%JAVA_HOME%\bin）和（%JAVA_HOME%\jre\bin）

5 Hello World

5.1 源代码

class HelloWorld{
    public static void main(String[] args){
        System.out.println("我的第一个Java程序，Hello World");
    }    
}

5.2 运行

1. javac HelloWorld.java
解释：虚拟机将java源文件编译成供机器能识别的class字节码文件

2. java HelloWorld

6 名词解释

1. 软件
有序排列的计算机数据与指令的集合

2. 程序
多个指令的有序排列

3. 代码
程序员用开发工具所支持的语言写出来的源文件

4. 指令
计算机执行某种操作的命令，它由一串二进制数码组成

5. 字节码
是一种包含执行程序，由一序列 op 代码/数据对组成的二进制文件，是一种中间码。字节是电脑里的数据量单位。

6. 平台
指的是程序运行的硬件或者软件环境，如：WindowsLinux、Unix、Mac OS

7. JIT
just-in-time compilation，即时编译器
